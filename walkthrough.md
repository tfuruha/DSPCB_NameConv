# ウォークスルー - PCBリネームツール

## 1. 準備
Pythonがインストールされている環境で実行してください。追加のライブラリインストールは不要です。

## 2. 動作テストの手順
1. `create_test_files.py` を実行します。
   ```bash
   python create_test_files.py
   ```
   - `test_data` フォルダが作成され、その中に変換前のダミーファイル群が生成されます。

2. `DSPCB_Renamer.py` を実行します。
   ```bash
   python DSPCB_Renamer.py
   ```
   - GUIウィンドウが表示されます。

3. 「ファイルを選択して実行」をクリックし、先ほどの `test_data` フォルダ内のファイルを **どれか1つ** 選択してください。

4. 処理結果を確認します。
   - すべて揃っていればリネームが完了します。
   - `test_data` フォルダの中身が `.gtl`, `.gts`, ... 等に変わっていることを確認してください。

## 3. 注意点
- プロジェクト名は `[プロジェクト名] - [層名].[拡張子]` という形式（半角スペース+ハイフン+半角スペース）で判別します。
- リネーム後のファイル名が既に存在する場合、上書きの確認ダイアログが表示されます。
